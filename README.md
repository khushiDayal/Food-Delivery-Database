# Food Delivery Database (MySQL)

A relational SQL project to analyze food delivery operations — customers, restaurants, delivery partners, and orders.

## Features
- Normalized database schema with foreign keys and constraints
- Analytical queries for delivery insights, repeat customers, and top cuisines

## How to Run
1. Open VS Code → Connect SQLTools → MySQL
2. Run `schema.sql` to create tables
3. Run `data.sql` to insert sample data
4. Run `analysis.sql` to view analytical outputs

## Tech Stack
MySQL | VS Code | SQLTools Extension
